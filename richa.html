Web technology refers to the tools, languages, protocols, and systems that create, deliver, and allow interaction with websites and web applications over the Internet. It includes technologies for both the visible "frontend"(like HTML, CSS, and JavaScript) and the server-side"backend" (like databases and server-side languages) to build dynamic and interactive online experiences.

Hypertext Markup Language

Hypertext Markup Language is the standard markup language for documents designed to be displayed in a web browser It defines the content and structure of web content. It is often assisted bt technologies such as Cascading Style sheets and scripting languages such as JavaScript.

CSS is a stylesheet languages used to describe the presentation and styling of a document written in a markup language such as HTML or XML. It is a fundamental technology of the World Wide Web, alongside HTML and JavaScript.
